# imageAnalysis package
This package contains basic image handling for now. The primary file can be used to view and extract metadata from .czi files. It can also be used to convert images obtained from such files to .tiff format.

Soon there will be a class (DCCImage) that will be used to have a uniform data structure to store image information. It will be the main object used for more complex image analysis like segmetation, filtering, etc.
